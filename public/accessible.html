◌ Global Accessibility Features

This entire website has been designed with accessibility in mind, including:

• Full Keyboard navigation support (Tab↹)
• Focus indicators for interactive elements
• High contrast design with few flashing colors.
• Screen reader compatibility, only text intended for reading is displayed as text.
